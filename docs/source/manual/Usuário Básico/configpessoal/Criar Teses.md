# Manual do Usuário Básico Super.GOV.BR 2.0 - Configuração do Perfil Pessoal - Criar Teses

## Disposições Gerais

O documento seguinte é uma divisão do Manual do Usuário Básico Super.GOV.BR 2.0. Para acessá-lo e ter maiores informações gerais sobre as funcionalidades do sistema, clique aqui.  

Uma das divisões do Manual do Usuário Básico é a Configuração do Perfil. Para acessá-lo e ter maiores informações gerais sobre as configurações do Usuário, clique aqui. 

Teses são blocos de palavras que podem ser colocados nas Minutas. Esses blocos de texto têm como intuito facilitar a adição de Jurisprudências e Legislações no editor, agilizando o trabalho dos Usuários. 

## Tela de Teses

A tela de Teses é acessada no Menu de Configuração do Perfil Pessoal. A Figura 1 abaixo é a Tela de Teses:

<img src="../../manual/figuras/Tela_Teses.png"/><p style="text-align: center;">*Figura 1 – Tela de Teses*</p>

Seguem indicadores dos botões (<ins>Figura 1</ins>):

1. Cria um registro de nova Tese;
2. Colunas. Por esta opção será possível gerenciar as colunas que serão disponibilizadas na tela;
3. Mostrar inativos. Opção que habilita mostra de Teses Ativas e Inativas;
4. Edita o cadastro de uma Tese;
5. Editar Conteúdo. Edita o conteúdo de uma Tese;
6. Remover. Exclui a Tese cadastrada;
7. Opção que permite configurar a quantidade de registros a serem mostrados em tela;
8. A tarja indica a quantidade de páginas relacionadas a consulta em tela. A navegação entre as páginas poderá ser feita por meio do uso das setas disponíveis.

## Como Criar uma Tese

Para criar uma nova Tese, o Usuário Básico deve seguir os seguintes passos no Super.GOV.BR 2.0:

1. Acessar a tela de Tese e clicar no botão "**Novo**" (indicado como 1 na Tela de Teses);
2. Em seguida, deverá preencher os campos disponíveis no formulário com as informações referentes a Tese a ser cadastrada;

``` {note}
Nota: Todos os campos marcados com * são de preenchimento obrigatório.
```

* **Ativo**: Flag de seleção obrigatória, destinada a indicação da atividade (caso selecionado) ou inatividade (caso desativado) da Tese;
* **Nome***: Campo destinado ao preenchimento do nome da Tese;
* **Descrição***: Campo destinado ao preenchimento da Descrição da Tese;
* **Modalidade***: Campo destinado ao preenchimento/seleção do tipo de Tese (Jurisprudência, Legislação, Tese).

3. Clicar em Salvar: a tela de Teses será acessada novamente;
4. Clicar em Editar Conteúdo (indicado como 5 na <ins>Figura 1</ins>): o editor de texto será aberto;
5. Inserir o conteúdo da Tese;
6. Clicar em Salvar.

Pronto, sua Tese está disponível para uso nas minutas.
